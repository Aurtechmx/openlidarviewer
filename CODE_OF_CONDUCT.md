# Code of Conduct

## Our pledge

We want OpenLiDARViewer to be a welcoming, harassment-free project for
everyone — regardless of background, identity, or experience level.

## Expected behavior

- Be respectful, considerate, and constructive.
- Welcome newcomers and differing viewpoints.
- Focus on what is best for the project and its community.
- Accept constructive feedback gracefully.

## Unacceptable behavior

- Harassment, insults, or discriminatory language or imagery.
- Personal or political attacks.
- Publishing others' private information without consent.
- Other conduct that would reasonably be considered inappropriate.

## Scope

This Code of Conduct applies in all project spaces — the repository, issues,
pull requests, and discussions — and whenever someone is representing the
project in public spaces.

## Enforcement

Report unacceptable behavior to **info@aurtech.mx**. Reports are reviewed
in private. Maintainers may remove, edit, or reject contributions, or take
other appropriate action.

This Code of Conduct is adapted from the
[Contributor Covenant](https://www.contributor-covenant.org/).
