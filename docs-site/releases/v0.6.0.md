# v0.6.0

The stable v0.6 release. Spatial operations are deterministic, explicit and
non-destructive: source geometry is immutable by proof, mounting is a
Float64 placement, measurements and layers explain their own context, and
streamed exports state their scope in the written file. Corrupt COPC input
fails with structured errors. Claims and stability are governed by
version-controlled policy (`CLAIMS_AND_LIMITATIONS.md`,
`STABILITY_POLICY.md`) enforced in the gate.

Full notes: [RELEASE_NOTES_v0.6.0.md](https://github.com/Aurtechmx/openlidarviewer/blob/main/RELEASE_NOTES_v0.6.0.md)
