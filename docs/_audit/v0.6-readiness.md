## 7. Suggestions carried to Phase 1 (from the three-lens review; none release-blocking)

Change detection / ICP: NaN-input guard in `icpRegister` (latent — sole caller
filters non-finite); correspondence trimming for partial overlap; `maxResidualM`
default `Infinity` contradicts the module header's refusal promise; stride sampling
can return ~2× `maxSamples`; `boxPoints` heap-boxes both clouds on the main thread
(move compare to a worker); the compare panel's dynamic imports are awaited outside
`try` (chunk-load failure leaves "one moment…" forever); `compareDtms` origin-offset
note prints source units as "m" for direct callers.

Integrity / capsule: dead scaffolding (`AuditLog`/`verifyAuditChain`,
`SessionFindings`, the never-populated `edits` + `sigma` manifest fields) whose
internal docs overpromise — wire in or trim; "computationally infeasible" comment
invites over-trust of an unkeyed digest (reword: detects edits made without
re-stamping); verifier "Produced by" is self-reported (label it); hidden file input
leaks on picker cancel; unhandled dynamic-import rejections around verifier/report
export; session-restored trust grades are un-digested (consider stamping the capsule
or softening the restore toast to "author-graded"); `pointCount` never threaded into
real reports; `canonicalize` ignores `toJSON` (guard against future `Date` fields).

PWA / SW: cache grows without bound across deploys (old hashed assets never pruned —
deliberate trade-off for stale-tab survival; add pruning against the current index);
`skipWaiting` silently skipped if one precache asset 404s; offline covers only
previously fetched lazy chunks (no messaging); hardcoded `/sw.js` registration breaks
sub-path deploys (use `BASE_URL`).

Editing / clip / snap: undo history bounded by count not bytes (50 whole-cloud swaps
can hold GBs — add a byte budget); terrain Analyse ignores the active clip while file
export respects it (disclose or filter); snapping can snap to clip-hidden /
class-hidden points (thread the same accept predicate picking uses); snap silently
dead under orthographic cameras (compute ortho px→world or disable the button); snap
index built synchronously at reveal (build lazily); export-path terrain compute is
uncancellable; `ClipPanel` accepts min > max silently.

Vocabulary watch (not violations of the stated forbidden list, but tightening
candidates): the green trust caption "Verified — well supported by measured points",
the "N verified" roll-up, and the "Evidence restored — N verified" toast — consider
"well-supported" phrasing in a Phase 1 pass (tests pin the current strings);
`measurementTrust.ts` "can't be certified as metres" is a negation (fine), could
avoid the word entirely. A repo-wide sweep found **zero** affirmative
"signature" / "survey-grade" / "certified" claims in user-facing text; disclaimers are
the dominant usage, and `terrainAssessment.ts` encodes the forbidden-claims rule.
